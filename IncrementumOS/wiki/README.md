# wiki/ — base de conhecimento do IncrementumOS

Esta pasta é o **contexto base** que os agentes (`incrementum_associate` e `incrementum_analista`) leem em toda sessão. É também o vault do Obsidian de Eduardo — a mesma pasta vista pelas duas pontas:

- **Eduardo** edita aqui via Obsidian (vault apontando para `IncrementumOS/wiki/`).
- **Agentes** leem os mesmos arquivos via filesystem em cada sessão.

## Estrutura

```
wiki/
├── philosophy/   # princípios inegociáveis, como Eduardo pensa
├── framework/    # critérios, processos, checklists
└── teses/        # teses de portfólio (evolutivo → Atlas quando maduras)
```

## Princípio

Tudo aqui é **evolutivo**. Não há filosofia "pronta" copiada de fora. Eduardo registra, os agentes incorporam, e iteramos juntos. Quando vazio, os agentes operam **sem assumir** estilo externo — perguntam ou explicitam ausência de contexto.
