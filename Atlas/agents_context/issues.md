# Atlas — issues.md

_Open issues raised by Associate (or by any agent flagging something that needs attention). CIO triages._

## Schema

```
### <issue_id> — <short title>
Opened: <YYYY-MM-DD> by <agent>
Severity: blocker | high | medium | low
Target: <wiki page or work queue entry the issue refers to>
Description: <1-3 sentences>
Resolution: <blank until resolved; then: date + one-liner + link to fix>
```

`blocker` means: nothing in the affected area proceeds until fixed.
`high` / `medium` / `low` are priority hints for the weekly triage.
